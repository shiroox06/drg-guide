🕹️ Cross Hotbar — Configuración Óptima

Pon tu nivel, valida las skills que tienes, y configura el hotbar exactamente como la guía lo recomienda para tu rotación actual.

🎚️ TU NIVEL: Lv1
🤲 Regla de Velocidad
⚡ R2 + Cara (△○□✕) — MÁXIMA VELOCIDAD
Anular derecho sostiene R2, pulgar derecho presiona la cara. Misma mano — cero delay. Solo para skills del combo principal que no requieren pensar.
🎯 R2 + Cruz (↑↓←→) — RÁPIDO CON VENTANA
Anular derecho en R2, pulgar izquierdo en cruz. Manos cruzadas — hay medio segundo. oGCDs frecuentes que no son combo encadenado.
🕐 L2 + Cara (△○□✕) — PLANIFICADO
Anular izquierdo en L2, pulgar derecho en cara. Manos separadas. Cooldowns largos del burst window que se planifican.
💡 L2 + Cruz (↑↓←→) — SITUACIONAL
Anular y pulgar izquierdo. Misma mano izquierda. AoE combo y utilities que no necesitan velocidad de ms.
🎮 Tu Hotbar
ℹ️ LAYOUT A — CUÁNDO USARLO
Tu layout por defecto. Contiene el combo principal, los oGCDs del burst window y el AoE combo completo. El 95% del combate ocurre aquí.
🔄 Presiona R1 para acceder a estas skills.
✅ Skills Disponibles a Tu Nivel
⚙️ Cómo Configurarlo en PS5
  • 1
    Abre el menú principal → System → Character Configuration → Hotbar Settings → Cross
  • 2
    Activa "Enable Cross Hotbar" si no está activado. Asegúrate de tener activado "Enable expanded cross hotbar" para acceder al segundo set con R1.
  • 3
    Cierra el menú. Sostén R2 — aparece el hotbar R2 en pantalla. Arrastra las skills desde tu lista de habilidades (Actions & Traits) a cada slot según la guía.
  • 4
    Haz lo mismo con L2 para el hotbar L2.
  • 5
    Presiona R1 para cambiar al Layout B. Repite el proceso para asignar las skills del Layout B en R2 y L2.
  • 6
    Para editar el hotbar desde el HUD: mantén presionado el botón Options durante el juego → entra al modo de edición del HUD → arrastra los slots.
💡 Tip: Configura primero solo los slots que tienes disponibles a tu nivel actual. Cuando llegues a un nivel nuevo y desbloquees una skill, vuelve a este módulo, pon tu nivel en el slider, y ve exactamente dónde agregarla.
