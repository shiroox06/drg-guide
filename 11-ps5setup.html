💡 El layout es una sugerencia — ajústalo a lo que te resulte cómodo. Lo crítico es que el combo principal (True Thrust, Vorpal, Full Thrust, Disembowel) esté en los 4 botones de acción principales sin modificador.
⚙️ Configuraciones Recomendadas
  • 🎯
    System → Character Configuration → Control Settings → GamepadCross Hotbar Display: Activa "Always Show" para ver siempre el hotbar sin tener que mantener presionado.
  • 🔔
    System → Character Configuration → UI SettingsCastbar + Enemy Castbar: Actívalos ambos. Ver el cast del boss es esencial para saber cuándo usar Feint.
  • 💊
    HUD LayoutPon el DoT tracker visible: Mueve la barra de status effects del objetivo a un lugar donde lo veas fácilmente — necesitas ver el timer de Chaotic Spring.
  • 🎮
    System → Character Configuration → Control SettingsConfirm Targeting: Activa "Prioritize Nearest Enemy" para que el joystick izquierdo no te cambie el target durante combate.
  • 🌊
    Character Configuration → Display Name SettingsShow own HP: Activa para ver tu HP sin necesitar mirar la party list completa.
📝 Macros Útiles

Los macros en FFXIV son simples — van en System → User Macros. Máximo 15 líneas por macro. En PS5 se ejecutan igual que cualquier skill en el hotbar.

MACRO 1 — Battle Litany con aviso al grupo:

/action "Battle Litany" /p Battle Litany! Burst window open! <se.3>

MACRO 2 — Feint con aviso al tank:

/action "Feint" <t> /p Feint on <t>! <se.1>

MACRO 3 — Aviso "first time" al entrar a dungeon:

/p Hi! First time here, please bear with me <se.2>
⚠️ No uses macros para el combo principal. Los macros tienen un delay de 1 segundo entre líneas — rompen la rotación. Solo son útiles para habilidades únicas con comunicación.
